# Project Name

A C program being able to open & modify then save images. 

## Features
Reading, Saving & Editing of BMP 8 & BMP 24 images.

## Installation & Usage

Clone the repo:  
`https://github.com/N-MTN/c-imagery`

Compile the C file:  
`gcc launcher.c -o launcher`

Run the program:  
`./launcher.c`
